# My Site 

A modern personal website built with Next.js, React, and TypeScript.

## Getting Started

First, install the dependencies (if you haven't already):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Introduction Page**: Showcases your background and focus on system architecture and distributed systems
- **Toggle Switch**: Click the switch to show/hide your projects section
- **Modern Design**: Dark theme with smooth animations and responsive layout

## Project Structure

- `app/page.tsx` - Main page component
- `app/layout.tsx` - Root layout
- `app/globals.css` - Global styles
- `package.json` - Dependencies and scripts

## Build for Production

```bash
npm run build
npm start
```
